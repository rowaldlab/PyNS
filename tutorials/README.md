## PyNS Tutorials

- To be able to run the tutorials from the `tutorials` folder, you need to compile the NEURON mechanism files (mod files) also from there:
    ```
    cd ./tutorials
    nrnivmodl ../src/pyns/mod_files/**/*.mod
    ```
